# 用户帮助中心 - 退换货指南

## 🌟 核心政策快读
* **不满意？** 7天内随便退（只要没坏）。
* **坏了？** 15天内包换，1年内保修。
* **发货慢？** 超过48小时未发货，自动赔付 5% 违约金。

## 💰 关于运费（谁来出钱？）
很多用户关心运费问题，我们制定了以下简明规则：

| 场景 | 谁出运费？ | 备注 |
| :--- | :--- | :--- |
| 商品有质量问题 | **商家承担** | 需提供照片/视频凭证 |
| 不喜欢/拍错了 | **用户承担** | 需保持包装完整 |
| 拒收快递 | **用户承担** | 将扣除双程运费 |
| 商家发错货 | **商家承担** | 请勿拆封原包装 |

## ⚠️ 特殊商品不支持退货列表
> 请注意，以下商品一经售出，除非质量问题，否则无法退换：
1.  **贴身衣物**：内裤、文胸、保暖内衣（为了大家的卫生安全）。
2.  **生鲜食品**：车厘子、海鲜等短保质期商品。
3.  **虚拟商品**：已发货的游戏激活码、会员充值。

## 📦 如何寄回商品？
1.  进入"我的订单" -> 点击"申请售后"。
2.  审核通过后，系统会提供退货地址。
3.  请在包裹内附上一张纸条，写明：**订单号、联系人电话、退货原因**。
